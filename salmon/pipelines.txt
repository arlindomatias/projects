## Índice Salmon (uma vez por dataset)
salmon index \
  -t gencode.vM35.transcripts.fa \
  -i salmon_mm39_index \
  -k 31 \
  --gencode

## Rodar Salmon em uma amostra
salmon quant \
  -i salmon_mm39_index \
  -l A \
  -1 fastq/SRR33247678_1.fastq \
  -2 fastq/SRR33247678_2.fastq \
  -p 8 \
  --validateMappings \
  --gcBias \
  --seqBias \
  -o salmon_quant/SRR33247678

less salmon_quant/SRR33247678/logs/salmon_quant.log 

Olhe especialmente:
- Detected lib type
- Mapping rate
- Warnings
Se a taxa de mapeamento for absurdamente baixa (<30%), pare aqui. Algo está errado.

## Rodar Salmon em loop
for r1 in fastq/*_1.fastq; do
  base=$(basename $r1 _1.fastq)
  r2=fastq/${base}_2.fastq

  salmon quant \
    -i salmon_mm39_index \
    -l A \
    -1 $r1 \
    -2 $r2 \
    -p 8 \
    --validateMappings \
    --gcBias \
    --seqBias \
    -o salmon_quant/$base
done

### IMPORTANTE: Caso o tamanho dos arquivos sejá maior que o espaço disponível,
é possível baixar e processar por partes. O Salmon não precisa dos FASTQs, 
apenas dos arquivos quanto.sf. Portanto, tendo o sf, é possível apagar FASTQs e sra, mantendo:
- salmon_quant/SRRxxxx/quant.sf
- metadados (seu sample_table)
- index

ls salmon_quant/SRR33247676
Agora você precisa ver:
- quant.sf
- aux_info
- libParams
- logs

Antes de qualquer DE:
awk '{s+=$4} END {print s}' salmon_quant/SRR33247677/quant.sf
A soma dos TPMs deve dar ~1.000.000 (com erro de arredondamento).

awk '{s+=$5} END {print s}' salmon_quant/SRR33247676/quant.sf
Isso deve dar algo da ordem de dezenas de milhões, coerente com o log.

# Estatística
Com poucos SRR, você não faz DE ainda, mas já pode:
- importar com tximport (Bioconductor)
- verificar correlação entre amostras
- testar PCA
- validar que o pipeline gera números estáveis
O ponto é: confirmar que tudo funciona antes de escalar.

## Decisão racional antes de baixar tudo

Depois do piloto, você olha:
- tempo por SRR
- uso de disco
- taxa de mapeamento
- variabilidade entre amostras

## Para o R
O que vem em seguida (próximo degrau lógico)
Depois que 2–3 SRRs quantificarem corretamente, o próximo bloco é:
- checar consistência entre SRRs
- importar com tximport
- decidir nível: transcript vs gene
- só então pensar em DE
