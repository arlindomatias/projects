# English
## Project directory
In this directory, my projects are organized into different folders according to their nature.  
Some directories are still under development.

### behavioral_analysis
This directory contains code used for behavioral analysis, including code (in R) for:
- calculation of behavioral indices and conversion of hexadecimal codes for file blinding in manual analysis;
- code for creating ethograms;
- code for multivariate analysis.

### salmon
Bash scripts for FASTQ data extraction and sample processing with Salmon.

### bulk_rna_seq
R scripts for bulk RNA-Seq data analysis.

### single_cell_rna_seq
Python scripts for single-cell data analysis.

### microbiome
R scripts for microbiome data analysis.

### latex
LaTeX scripts for document creation.

### machine_learning
Python code for machine learning applications.

### web
Code for creating simple websites (HTML/CSS/JavaScript) and for web scraping (Python).
